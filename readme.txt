PayU Paisa Payment Gateway for CampTix plugin

== Description ==

PayU Paisa (by PayU India) Payment Gateway for the CampTix plugin.

Accpet payments in INR through PayU Paisa using the CampTix plugin. CampTix plugin needs to be installed and activated for the PayU Paisa payment gateway to work.

== Installation ==

1. Upload `camptix-payupaisa-gateway` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to `Tickets -> Setup` in your WordPress admin area to set the currency to INR and activate the PayU Paisa gateway.
